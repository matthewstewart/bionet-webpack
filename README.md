# Bionet UI
A Webpack Build That Aliases React Code Into Preact. Bulma & Sass Support Included.


## Install

### Clone
```
git clone https://github.com/matthewstewart/bionet.git
cd bionet
```

### Install Dependencies
```
npm install
```

### Development Server
```
npm run start
```

### Production Build
```
npm run build
```
